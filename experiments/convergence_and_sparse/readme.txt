This folder contains all necessary codes for testing the convergence and sparse interaction abilities of SIES

-Convergence ability comparsion
  -convergence_time_sies.py: This script records the convergence time of SIES to the WALK, BOUND, and TROT gaits across 1000 random initial conditions.
  -convergence_time_salamander.py This script records the convergence time of the Salamander model to the WALK, BOUND, and TROT gaits across 1000 random initial conditions.
  -convergence_time_diffusive.py This script records the convergence time of the diffusive model to the WALK, BOUND, and TROT gaits across 1000 random initial conditions.
  -convergence_time_fc.py This script records the convergence time of the FC model to the WALK, BOUND, and TROT gaits across 1000 random initial conditions.

- Sparse interaction
  -sparsity_test_small_world.py: This script records the phase metrics of randomly generated synchronization modes produced by SIES on small-world networks with varying sparsity levels (k_n ranging from 2 to N-2 in steps of 2, p_r = 0). The network scale is adjustable.
  -small_world_test: This script records the phase metrics of arbitrary synchronization modes generated by SIES on small-world networks with fixed k_n = 0.3 and varying p_r > 0. For each p_r, 20 random small-world networks are generated. Both network scale and p_r are adjustable.
