This folder contains the necessary codes for generalization experiments of SIES
-trav_states.py: This script records the node states of the CDS under the traveling-wave setting, with adjustable network scale and intrinsic node dynamics.
-trav_metrics.py: This script records the waveform metrics of the CDS network with 4–120 units when the target mode is traveling wave, under different intrinsic node dynamics. Metrics include the target-aligned order parameter and phase RMSE.
-trav_convergence: This script records the convergence time to the traveling wave for CDS networks with 4–120 units across different intrinsic node dynamics.
-random_phase_metrics: This script records the waveform metrics of CDS networks with 4–120 units when the target mode is arbitrary, across varying intrinsic node dynamics. Metrics include the target-aligned order parameter and phase RMSE.
