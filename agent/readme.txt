This folder contains models and RL algorithms of the RL agent.
- actor_net.py contains the basic implementation of SIES-like coupling mechanisms.
- networks.py contains the relevant actor and critic networks that used in RL training and evaluations
- TD3.py contains the Twin Delayed DDPG (TD3) algorithm with specific adaptation to GNN policies.
- replay_buffer.py constructs the replay buffer used in off-policy RL algorithm.
